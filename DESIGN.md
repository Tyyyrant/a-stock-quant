# 量化与技术分析系统 — 架构设计文档 v3

## 1. 核心理念

不是简单的条件规则筛选，而是**多因子 Alpha 模型 + 风控约束**的量化投研体系。

参考框架：
- **Barra 风险模型**（MSCI CNE5）：将收益分解为共性因子收益 + 特质收益
- **Fama-French 五因子**：市场、规模、价值、盈利、投资
- **幻方量化实践**：数百个 Alpha 因子 + 深度学习非线性组合 + 严格风控

## 3. 因子体系设计

这是整个系统的核心。按 Barra 的分类框架，组织 A 股有效的因子。

### 3.1 因子全景

```
                    因子分类
                       │
        ┌──────────────┼──────────────┐
        │              │              │
    Barra 风格因子   A股特异因子   技术形态因子
    (风险分解)      (Alpha来源)    (你关注的结构)
        │              │              │
    · 规模 SIZE      · 反转 REV     · 底部突破
    · 价值 VALUE     · 低波 LOVOL   · 倍量阳线
    · 动量 MOMENTUM  · 资金流 FLOW  · 均线发散
    · 波动 VOL       · 情绪 SENT    · 关键位突破
    · 质量 QUAL      · 筹码 CHIP
    · 成长 GROWTH    · 龙虎榜 LHB
    · 流动性 LIQ
```

### 3.2 Barra 风格因子（风险端）

这些因子不直接产生 Alpha，但用于**控制风险暴露**和**做行业中性化**。

#### SIZE — 规模因子

```
定义: 对数市值 ln(market_cap)

A股实践:
  - 小盘溢价长期存在，但注册制后逐渐减弱
  - 大小盘风格切换明显（每年1-2次）
  - 当前风格判断: 对比中证2000 vs 沪深300 近20日走势

暴露计算:
  SIZE_exposure = ln(market_cap) 的截面标准化 Z-score

风控用途:
  - 不想赌风格 → 约束组合 SIZE 暴露在 ±0.3 标准差内
  - 明确看多小盘 → 允许 SIZE 暴露偏负（-1.0 ~ -0.3）
```

#### VOL — 波动率因子

```
定义: 
  - 日收益率标准差 (过去 20/60 个交易日)
  - CAPM Beta (个股对中证全指回归)
  - 特质波动率 (残差波动率)

A股实践:
  - A股低波异象显著：低波动股票长期跑赢高波动
  - 这是A股最稳定的Alpha来源之一
  - 低波动 ≠ 低价，低波因子在下跌市中保护最好

信号方向:
  - 做多低波 + 做空高波 → 稳定正收益
  - 我们的使用：VOL得分越低（波动越小），因子分越高
```

#### MOMENTUM — 动量因子

```
定义:
  - 短期动量: 过去1个月收益（剔除最近5天，避免短期反转干扰）
  - 中期动量: 过去6-12个月收益（剔除最近1个月）
  - 残差动量: 剔除市场和行业收益后的特质动量

A股实践:
  - A股短期反转效应 > 中期动量效应（和美股相反）
  - 1-5日反转极强（散户追涨杀跌制造）
  - 6-12月有弱动量
  - 幻方的做法: 分段处理——短期做反转，中期做动量

信号方向:
  - RSTR_5d (5日反转): 跌得多 → 高分（反转买入）
  - RSTR_1m (1月动量): 涨得多 → 高分（顺势持有）
  - RSTR_6m (6月动量): 涨得多 → 高分
```

#### VALUE — 价值因子

```
定义:
  - EP: 净利润 / 总市值（1/PE）
  - BP: 净资产 / 总市值（1/PB）
  - CFP: 经营现金流 / 总市值
  - DP: 股息率

A股实践:
  - 价值因子在A股波动大，单用效果差
  - 但可以过滤泡沫: EP过低（PE过高）的票有估值风险
  - 价值+质量组合效果好（便宜且好的公司）

信号方向:
  - 做风控过滤用: EP < 0（亏损股）→ 降分或剔除
  - 行业相对估值: 同行业内EP越高越好
```

#### QUAL — 质量因子

```
定义:
  - ROE: 净资产收益率（近12个月）
  - ROE稳定性: ROE过去5年的标准差（越小越稳定）
  - 杠杆率: 资产负债率
  - 应计利润占比: (净利润 - 经营现金流) / 总资产

A股实践:
  - ROE是最有效的质量指标
  - ROE > 15% 且稳定 > 3年 → 长期牛股集中营
  - 高杠杆在A股是减分项（信用风险）

信号方向:
  - ROE越高越好，但 > 30% 要警惕（可能是非经常性收益）
  - 杠杆率越低越好
  - 应计利润占比越低越好（利润质量高）
```

#### GROWTH — 成长因子

```
定义:
  - 营收增长率 (YoY, 近1年/3年)
  - 净利润增长率 (YoY, 近1年/3年)
  - 分析师一致预期变化 (EPS_Fwd Revision)

A股实践:
  - 成长因子在牛市中表现最好
  - 预期修正因子（分析师上修/下修）短期效果显著
```

#### LIQ — 流动性因子

```
定义:
  - 日均换手率 (近20日)
  - 日均成交额 (近20日)
  - Amihud非流动性指标: |日收益| / 日成交额

A股实践:
  - 低换手率股票长期跑赢（和直觉相反）
  - 异常高换手 → 可能是出货或对倒
  - 流动性太差（日均成交 < 2000万）→ 剔除，无法进出

风控用途:
  - 强制剔除日均成交额 < 3000万的股票
  - 单票仓位上限 = 日均成交额的 1%（保证一天能卖完）
```

### 3.3 A股特异 Alpha 因子

这些是量化机构在 A 股实际使用、有效性经过验证的因子。

#### 短期反转因子 (REV_5d)

```
定义: 过去5个交易日的累计收益，取负值

逻辑: A股散户占比高，过度追涨后必然回调，过度杀跌后必然反弹。
      5日反转是A股最强的短期Alpha来源。

计算:
  REV_5d = -(过去5日涨跌幅)

用法: 不单独使用——配合趋势过滤（MA20之上用反转，MA20之下不用）
```

#### 低波异象因子 (LOWVOL)

```
定义: 近60日日收益率标准差的倒数

逻辑: A股低波动股票长期跑赢，这是最稳健的因子之一。

注意: 低波因子在牛市中跑输（牛市中高beta更好），需要结合大盘温度动态调整权重：
  - 大盘偏弱/震荡 → 低波因子权重加倍
  - 大盘强势 → 低波因子权重减半
```

#### 北向资金因子 (NORTH_FLOW)

```
定义:
  - 北向持仓占比变化（月度）
  - 北向连续流入天数
  - 北向持仓市值 × 变化幅度

逻辑: 北向资金是A股最聪明的资金之一，持续加仓的股票后续大概率跑赢。

数据源: market_data.json → north_flow
```

#### 融资融券因子 (MARGIN)

```
定义:
  - 融资余额变化率（近5日）
  - 融资买入额占总成交比
  - 融券余额变化（做空力量）

逻辑:
  - 融资余额温和上升 → 看多力量积累（利好）
  - 融资余额急剧上升 → 情绪过热（利空，反向指标）
  - 融资买入占比 > 20% → 过度投机，减分

注意: 融资因子是滞后指标，需要配合价格走势看。
```

#### 龙虎榜因子 (LHB)

```
定义:
  - 龙虎榜净买入额
  - 机构席位 vs 游资席位占比
  - 买方/卖方金额比

逻辑:
  - 机构席位净买入 → 最强看多信号
  - 游资席位净买入 → 短期可能有波动，持续性差
  - 机构+游资同向 → 共振信号
  - 纯游资、无机构 → 可能是对倒，谨慎
```

#### 资金流因子 (MONEY_FLOW)

```
定义:
  - 主力资金净流入（超大单+大单 - 中小单）
  - 资金流连续流入天数
  - 资金流强度: 净流入 / 日均成交额

逻辑:
  - 连续3天主力净流入的股票，后续5日跑赢概率 > 60%
  - 但要区分"真流入"还是"对倒"——配合价格看：
      价涨量增 + 主力流入 = 真
      价平量增 + 主力流入 = 小心对倒出货
```

#### 分析师一致预期因子 (ANALYST)

```
定义:
  - 分析师覆盖家数变化（新覆盖 → 利好）
  - EPS预测上修/下修比例
  - 目标价上修幅度

逻辑:
  - 分析师的行为比结论更有信息量:
      新覆盖 > 上调评级 > 上调EPS > 上调目标价
  - 卖方分析师普遍乐观，所以要关注"变化"而非"绝对值"
```

#### 股东/高管行为因子 (INSIDER)

```
定义:
  - 高管增持/减持
  - 重要股东增减持
  - 回购公告

逻辑:
  - 高管增持（自有资金）→ 最强内部看多信号
  - 高管减持可能是流动性需求（不一定是利空）
  - 公司回购注销 → 强利好
  - 但这些信号发生频率低，覆盖率有限
```

#### 行业景气度因子 (SECTOR_MOMENTUM)

```
定义:
  - 行业近20日涨幅排名百分位
  - 行业内个股涨跌比
  - 行业成交额占比变化

逻辑: A股行业轮动特征明显。处于涨幅排名前30%的行业，未来一周继续跑赢的概率约55%。

用法: 这是Layer 2板块共振的数据基础。
```

### 3.4 技术形态因子（你关注的核心）

技术形态不是主观看图，而是**量化为可回测的规则**。

#### 底部倍量突破因子 (BOTTOM_BREAKOUT)

```
定义: 
  score = 4个维度得分的乘积

维度1 — 超跌程度（权重 0.25）
  近20日跌幅: -30%→100分, -20%→80分, -10%→50分, >0%→0分

维度2 — 底部质量（权重 0.25）
  横盘天数: >15天→100分, 10-15天→80分, 5-10天→50分, <5天→0分
  横盘振幅: <10%→100分, 10-15%→70分, >15%→30分
  取两者平均

维度3 — 爆发力（权重 0.30）
  量比: >3倍→100分, 2-3倍→80分, 1.5-2倍→50分, <1.5→0分
  涨幅: >7%→100分, 5-7%→80分, 3-5%→50分, <3%→0分
  K线质量: 光头阳→100分, 短上影→80分, 长上影→30分
  取三者平均

维度4 — 位置确认（权重 0.20）
  收盘站上MA20→100分, 站上MA10→60分, 仍在MA10下→0分
  
最终得分: 加权求和 → 标准化为 Z-score
```

#### 关键位突破因子 (KEY_BREAKOUT)

```
识别要突破的"关键位":
  - MA60/MA120/MA250
  - 近60日最高价
  - 前期密集成交区（20日均量最大的价格区间）

突破有效性评分:
  量比 > 2: +30分
  收盘站稳（非盘中突破又回来）: +25分
  突破后连续2日不跌回: +25分
  光头阳突破: +20分
  
假突破识别（扣分）:
  盘中突破后收盘跌回 → -50分
  突破量比 < 1.2 → -30分
  突破后次日就跌回 → 标记为假突破
```

#### 均线粘合发散因子 (MA_SQUEEZE)

```
粘合度: MA5/10/20 三条线的标准差 / 收盘价
  粘合度 < 1%: 极度收敛（即将爆发） → 100分
  粘合度 1-2%: 比较收敛 → 80分
  粘合度 2-3%: 一般收敛 → 50分

发散方向: 
  向上发散（MA5>MA10>MA20形成） → ×1.0
  向下发散 → ×0.0（废弃此信号）

爆发确认:
  放量（量比 > 1.5）→ +10分
  涨幅 > 3% → +10分

粘合时间权重:
  粘合 > 15天 → ×1.3（蓄势越久爆发越强）
  粘合 5-15天 → ×1.0
  粘合 < 5天 → ×0.7
```

#### MACD零轴金叉因子 (MACD_GOLDEN)

```
位置评分:
  零轴 ±1% 范围内金叉 → 100分（最标准的多空转换）
  零轴下方但 < 5% → 80分（从弱转强）
  零轴上方 → 60分（趋势延续，不是转折）
  零轴下方 > 10% → 30分（只是超跌反弹，非趋势反转）

金叉角度（DIF斜率变化）:
  DIF 加速上行（DIF_t - DIF_t-1 > DIF_t-1 - DIF_t-2）→ +10分
  DIF 匀速上行 → +5分

背离加分（稀有但强力）:
  股价新低 + MACD 未新低（底背离） → +20分
```

#### 缩量回踩因子 (PULLBACK)

```
前提: 前期存在有效突破（突破时量比 > 1.5 + 收盘站稳）

缩量程度:
  回踩量 < 突破量的 50% → 100分（抛压极轻）
  回踩量 < 突破量的 70% → 70分
  回踩量 > 突破量 → 20分（放量下跌，不是回踩是破位）

支撑确认:
  回踩不破突破位（或MA20）→ +20分
  企稳信号（收阳线 + 量比 < 0.6）→ +15分

警告:
  回踩破了突破位 → 形态失效，不计分
```

## 4. 市场环境诊断 (`market_diagnostic.py`)

在做 Alpha 选股之前，先判断当前市场处于什么状态——这决定了**各因子的权重应该怎么调**。

### 4.1 波动率区间 (Volatility Regime)

```
低波区间（沪深300 20日波动率 < 15%）:
  → 低波因子 + 质量因子 权重加大
  → 反转因子 权重减小（低波环境不适合做反转）

正常区间（15-25%）:
  → 各因子均衡配置

高波区间（> 25%）:
  → 反转因子 + 低波因子 权重加大
  → 动量因子 权重减小（高波动下趋势不可靠）
  → 仓位系数整体降低
```

### 4.2 风格诊断 (Style Regime)

```
大小盘风格:
  if 中证2000(20日) > 沪深300(20日) → 小盘风格
     → SIZE因子允许偏小盘（不做约束）
  else → 大盘风格
     → SIZE因子约束在 0-1 区间

价值/成长风格:
  if 国证价值(20日) > 国证成长(20日) → 价值风格
     → VALUE因子权重 × 1.5
  else → 成长风格
     → GROWTH因子权重 × 1.5
```

### 4.3 市场温度

```
温度 = 加权(指数位置、宽度、量能、资金、情绪、结构) / 100

不用来剔除股票，而是用来调整:
  - 因子权重
  - 仓位系数
  - 止损松紧度（弱势市场止损更紧）
```

## 5. 因子组合与打分 (`factor_compositor.py`)

### 5.1 因子标准化

```
所有原始因子值 → Z-score 标准化（截面）：

  Factor_Z[i] = (Factor[i] - mean(Factor)) / std(Factor)

然后去极值（Winsorize）：
  Z > 3 → 截断为 3
  Z < -3 → 截断为 -3
```

### 5.2 行业中性化

```
为什么要做：A股行业分化严重，不中性化的话选出来的全是热门行业股票。

做法：对每个因子 Z-score 做行业回归，取残差作为"纯因子暴露"：

  Factor_Neutral[i] = 残差( Factor_Z[i] ~ 行业哑变量 )

这样就剔除了"因为处于好行业所以因子值高"的部分。
```

### 5.3 因子权重（动态调整）

```
默认权重（正常市场）:
  技术形态因子: 25%   ← 你关注的结构
  反转因子:    15%   ← A股最强短期Alpha
  低波因子:    12%
  北向资金:    12%
  质量因子:    10%
  资金流因子:   8%
  动量因子:     8%
  价值因子:     5%
  分析师预期:   5%

大盘弱势时调整为（防守模式）:
  低波因子:    20%   ← 加大防御
  质量因子:    18%   ← 加大防御
  北向资金:    15%
  技术形态:    15%   ← 弱市突破容易假，降权
  反转因子:    10%   ← 弱市反转容易失败
  其余:      调低
```

### 5.4 综合评分

```
Alpha最终得分 = Σ( Factor_Z_Neutral[i] × Weight[i] × Direction[i] )

其中 Direction[i] 是因子方向（+1=因子值越大越好, -1=越小越好）

然后再次标准化为 0-100 分数，方便直观理解。

共振加分（解决你关心的"多维度共振"问题）:
  if 技术形态分 > 0.5 AND 北向因子分 > 0 AND 低波因子分 > 0:
      最终分 × 1.1 （多维确认，加10%）
```

## 6. 风控层 (`risk_controller.py`)

选出来的股票不能直接买，要过风控。

### 6.1 一票否决

| 条件 | 原因 |
|------|------|
| ST / *ST | 退市风险 |
| 近5日有过跌停 | 流动性风险 |
| 日均成交额 < 3000万 | 无法进出 |
| 近20日涨幅 > 50% | 不追高 |
| ROE连续2年为负 | 盈利能力差 |
| 有未了结的立案调查 | 合规风险 |
| 大股东高比例质押 (>70%) | 爆仓风险 |

### 6.2 Barra 风险控制

```
单票层面:
  - 不对任何单票下注过重：单票 Alpha 得分 > 3σ 时截断

行业层面:
  - 单个行业（申万一级）的股票数不超过总池的 30%
  - 避免行业过度集中
  
风格层面:
  - SIZE因子总暴露在 ±0.5 范围内（不赌风格）
  - 除非风格诊断明确指示，否则不偏
```

### 6.3 仓位管理

```
仓位 = 基础仓位 × 大盘温度系数 × 波动率系数

基础仓位: 由 portfolio.yaml 设定
大盘温度系数: 温度/100
波动率系数: 15% / 当前波动率（波动越大仓位越小）
```

## 10. 核心设计决策

1. **因子模型取代规则筛选**：技术形态也量化为因子，和其他因子平等竞争，IC 分析会告诉你哪个因子真正有效
2. **行业中性化**：不做的话选出来的全是热门赛道股，行业贝塔冒充了 Alpha——这是业余量化最常犯的错误
3. **动态权重而非固定权重**：市场状态变了，因子权重跟着变——低波因子在弱势市场加权重，在强势市场减权重
4. **Barra 做风控，Alpha 做收益**：Barra 管住风险不跑偏，Alpha 因子专注找超额
5. **所有规则可回测**：因子公式、权重、阈值全部 yaml 化，回测引擎遍历参数空间找最优
6. **大盘温度不再做闸门**：v2 中大盘 < 40 直接空仓太粗暴了。现在大盘温度用来调权重和仓位，不直接拦截选股——因为弱市中的超跌反弹恰恰是最肥的利润来源
